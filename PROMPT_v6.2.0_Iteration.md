# VisePanda v6.2.0 迭代提示词

> 目标：从"规划工具"升级为"一站式入境管家"，重构信息架构

---

## 一、架构调整

### Tab 合并

将当前 6 个 Tab 合并为 **3 个核心 Tab**：

| 新 Tab | 包含内容 | 说明 |
|--------|----------|------|
| **Chatbot** | AI 咨询 + 行程规划 + 城市问询 | 原 Ask + Plan + Cities 合并 |
| **Dashboard** | 所有功能聚合仪表盘 | 原通知工具、行程管理、酒店预订、团购、地图、城市浏览、设置等全部聚合 |
| **Translation** | 语音翻译 + 文字翻译 + 景点/餐饮/文化短语库 | 原翻译面板，新增数据库翻译 |

> 注意：删除原 Plan、Cities、Tools、Trips 独立 Tab，它们的功能全部汇入 Dashboard。

---

## 二、功能开发

### 1. 确认 DeepSeek 联通性

- 在 Chatbot 启动时做一次 DeepSeek API 健康检查
- 如果 API 不可用，显示明确的错误提示 + 备用方案
- 后端 `/api/health` 端点增加 DeepSeek 联通性校验

### 2. 高德地图 API 接入

- 申请/配置高德地图 JS API key（Web端，Key 放后端代理不暴露前端）
- 在 Dashboard 中嵌入地图组件：
  - 显示用户当前所在城市（IP 定位）
  - 显示附近酒店标记
  - 显示附近餐厅/景点标记
  - 支持地点搜索（中文输入 → 显示英文信息）
- 后端新建 `/api/maps` 模块：`api/maps.py`
  - `GET /api/maps/geocode?q=北京故宫` → 返回经纬度
  - `GET /api/maps/place?type=hotel&lat=...&lng=...` → 返回附近POI
  - `GET /api/maps/translate?name=故宫` → 返回英文名 + 拼音

### 3. 酒店预订 API 端口

- 后端新建 `/api/hotels` 模块：`api/hotels.py`
- 功能接口（先做接口定义和桩函数，具体对接后续完善）：
  - `GET /api/hotels/search?city=北京&checkin=2026-07-01&checkout=2026-07-03` → 返回酒店列表
  - `GET /api/hotels/detail?id=xxx` → 返回酒店详情（含外国人接待资质、英文服务、外卡支付）
  - `POST /api/hotels/book` → 预订（暂为记录意向，实际对接后续）
- 每个酒店需标注：
  - ✅ 接待外国人
  - ✅ 有英文服务
  - ✅ 收外卡 (Visa/Mastercard)
  - 距最近地铁站距离

### 4. 美团/团购 API 端口

- 后端新建 `/api/deals` 模块：`api/deals.py`
- 功能接口（先做接口定义和桩函数）：
  - `GET /api/deals/search?city=北京&type=food` → 返回团购/优惠列表
  - `GET /api/deals/detail?id=xxx` → 返回团购详情
  - 每个团购需标注外国人可用性：是否接受外卡支付、是否有英文界面引导
- 数据来源后续对接美团开放平台或大众点评

## 三、Dashboard 重设计

### 布局

将所有功能聚合到一个**多信息仪表盘**，采用卡片式布局：

```
┌─────────────────────────────────┐
│   🌤️ 上海 · 28°C  ☁️           │  ← 天气 + 定位
│   欢迎回来，[用户]               │
├──────────┬──────────┬───────────┤
│  🏨       │  🗺️      │  🛍️       │
│  酒店搜索   │  地图     │  团购      │
│  正在热的  │  附近POI  │  优惠      │
├──────────┴──────────┴───────────┤
│  📋 我的行程                      │  ← 已保存行程列表
│  ┌─ 北京3日游 ─────────────────┐ │
│  │ 06.28 - 06.30 · 3天        │ │
│  │ 故宫·长城·烤鸭               │ │
│  └────────────────────────────┘ │
├─────────────────────────────────┤
│  🏙️ 推荐城市                     │  ← 原城市卡片
│  [北京] [上海] [成都] [西安]     │
├─────────────────────────────────┤
│  🛠️ 旅行工具                     │  ← 原工具卡片
│  签证 · 支付 · 通讯 · 打包       │
└─────────────────────────────────┘
```

### 设计原则

- 信息密度适中，不堆砌
- 每个功能模块可折叠/展开
- 按使用频率排序：高频功能靠前
- 移动端单列滚动，桌面端多列网格
- 保持 English-native，中文括号标注

### 包含的功能模块

| 模块 | 内容 |
|------|------|
| 天气+定位 | IP定位显示当前城市 + 天气 |
| 酒店搜索 | 快捷搜索框 + 推荐/热门酒店 |
| 地图 | 高德地图组件，显示附近 POI |
| 团购优惠 | 附近团购/优惠推荐 |
| 我的行程 | 已保存行程列表，快捷入口 |
| 推荐城市 | 城市浏览入口 |
| 旅行工具 | 签证/支付/通讯/打包等工具入口 |
| 快速操作 | 常用功能快捷入口：翻译、叫车、问路 |
