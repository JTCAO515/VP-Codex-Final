## P1 设计

### 架构变更

```
现状:
  detect_intent (keyword) → extract_slots (LLM slot only) → decide_next_action (rule) → _render_reply (hardcoded)

v2.4:
  LLM 单次调用 → {intent, slots, reply, itinerary, actions}
  
新函数:
  llm_chat_orchestrate(text, prior_state) → OrchestrateResult
```

### LLM 接口契约

```python
@dataclass
class OrchestrateResult:
    intent: str          # plan | book_hotel | rfp
    reply: str           # 自然语言回复
    slots: dict          # 抽取的 slot
    confidence: dict     # 置信度
    ask: dict | None     # 追问
    itinerary: dict | None  # 生成行程时非空
    actions: list[dict]  # 额外 action
```

### 环境变量

```
LLM_ENABLED=1            # 启用 LLM（不启用则回退规则版）
LLM_BASE_URL=https://api.nuwaflux.com/v1
LLM_API_KEY=<key>
LLM_MODEL=<model>
```
