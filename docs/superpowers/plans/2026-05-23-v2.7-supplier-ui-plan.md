# v2.7 供应商后台 — 实现计划

> 状态: P0 对齐 ✅

**目标**: 供应商可登录、查看 RFP、提交报价、管理订单
**范围**: 供应商登录页 + 后台仪表盘 + 报价管理 + 订单管理
**不做**: 供应商注册审核流程、高级数据分析
**成功标准**: 供应商登录→查看 RFP→提交报价→查看订单→完成订单，测试 ≥ 26

## P1 设计

### 页面

| 路由 | 说明 |
|------|------|
| `/supplier/login` | API Key 登录 |
| `/supplier/dashboard` | 仪表盘：待报价 RFP、进行中订单 |
| `/supplier/rfps/{id}` | RFP 详情 + 报价表单 |
| `/supplier/orders` | 订单列表 |

### API（已有 + 新增）

| 已有 | 新增 |
|------|------|
| POST /suppliers | GET /supplier/me（验证 API key） |
| POST /suppliers/{id}:enable | GET /supplier/rfps（供应商视角 RFP 列表） |
| POST /rfps/{id}/quotes | PUT /quotes/{id}（修改报价） |
| | POST /quotes/{id}:withdraw（撤回） |
| | GET /supplier/orders（供应商订单列表） |
| | POST /orders/{id}:complete |

### 前端文件

| 文件 | 说明 |
|------|------|
| `frontend/supplier/login.html` | API Key 登录 |
| `frontend/supplier/dashboard.html` | 仪表盘 |
| `frontend/supplier/rfp.html` | RFP 详情 + 报价 |
| `frontend/supplier/orders.html` | 订单管理 |
| `frontend/supplier/app.js` | 共享 JS |
